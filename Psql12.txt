How to Uninstall PostgreSQL Silently

Open an Elevated Command Prompt by Right-Clicking on Command Prompt and select Run as Administrator

Enter the following command for PostgreSQL 12:

"%ProgramFiles%\PostgreSQL\12\uninstall-postgresql.exe" --mode unattended --unattendedmodeui none
